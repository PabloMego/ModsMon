# Ejecuta y despliega tu aplicación AI Studio

Este repositorio contiene todo lo necesario para ejecutar la aplicación localmente.

Ver tu app en AI Studio: https://ai.studio/apps/drive/1UY5YULN-cPIHWWNNCROGdGmAOegPzA_U

## Ejecutar localmente

**Requisitos:** Node.js

1. Instala dependencias:
   `npm install`
2. Configura `GEMINI_API_KEY` en [.env.local](.env.local) con tu clave de Gemini
3. Ejecuta la app:
   `npm run dev`
